# API manage nomad jobs app

### Connection with MySQL database
Create .env file
Add string with your parameters
```MYSQL_URL=admin:password@tcp(127.0.0.1:3306)/nomad_jobs```

### Getting Started

Download ZIP from Github  
`go run main.go`

### Validate Nomad job
`http://localhost:8000/validate`
